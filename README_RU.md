**Aegis** — это мощный и легкий плагин для управления регионами, разработанный специально для серверов **Allay**. Он предоставляет интуитивно понятный графический интерфейс (GUI) и систему команд для защиты вашего мира.

### ✨ Возможности
- **Защита регионов**: Запрет на разрушение, установку блоков и взаимодействие внутри регионов.
- **Управление через GUI**: Удобные меню для настройки регионов без запоминания сложных команд.
- **Визуальное выделение**: Используйте топорик (Деревянный топор) для легкого выделения углов региона.
- **Мультиязычность**: Полная поддержка английского (`en_US`) и русского (`ru_RU`) языков.
- **Производительность**: Оптимизирован для Allay для обеспечения максимальной скорости работы.

### 🎮 Команды
| Команда | Алиас | Описание |
|---------|-------|-------------|
| `/aegis menu` | `/rg menu` | Открывает главное меню управления. |
| `/aegis wand` | `/rg wand` | Выдает топорик для выделения региона. |
| `/aegis create <имя>` | `/rg create` | Создает регион по выделенным точкам. |
| `/aegis delete <имя>` | `/rg delete` | Удаляет существующий регион. |
| `/aegis info` | `/rg info` | Выдает инструмент проверки регионов. |
| `/aegis flag <регион> <флаг> <значение>` | `/rg flag` | Устанавливает флаг для региона. |
| `/aegis addmember <регион> <игрок>` | `/rg addmember` | Добавляет участника в регион. |
| `/aegis removemember <регион> <игрок>` | `/rg removemember` | Удаляет участника из региона. |

### 🛠️ Использование
1.  **Получите топорик**: Введите `/aegis wand`.
2.  **Выделите область**: 
    -   **ЛКМ (Левый клик)** по блоку для установки Точки 1.
    -   **ПКМ (Правый клик)** по блоку для установки Точки 2.
3.  **Создайте регион**: Введите `/aegis create МойРегион` или используйте меню.
4.  **Готово!** Ваша территория теперь защищена.

### 🚩 Флаги
Aegis поддерживает различные флаги для настройки защиты региона:
- `build`: Разрешить/Запретить строительство (По умолчанию `false` для чужаков).
- `pvp`: Разрешить/Запретить PvP.
- `entry`: Разрешить/Запретить вход.
- `exit`: Разрешить/Запретить выход.
- `chest-access`: Разрешить/Запретить доступ к сундукам.
- `use`: Разрешить/Запретить использование механизмов.
- `chat`: Разрешить/Запретить писать в чат.

### 🔐 Права (Permissions)
| Право | Описание |
|-------|----------|
| `aegis.admin` | Полный доступ ко всем командам и обход защиты. |
| `aegis.flag.<name>` | Право на изменение конкретного флага (например, `aegis.flag.pvp`). |
